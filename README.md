# Football API

## Technologies used

-   [Node.js](https://www.nodejs.org)
-   [Express](https://www.expressjs.org)
-   [Sequelize](https://www.sequelize.org)
-   [PostgreSQL](https://www.postgresql.org)

## Installation

```
> git clone git@github.com:ducatrinh/football-api.git
> cd football-api
> npm install
> node index.js
```
